# Compatibility Notes

These notes describe the behavior Rewair needs to preserve while replacing the
EMW3165 radio-module firmware.

## Files

- `uart_protocol.md`: F103 <-> EMW3165 frame format, command vocabulary, and
  local bridge scope.
- `radio_platform.md`: EMW3165/WICED platform notes, repo map, build/flash
  paths, and the current UART investigation thread.

For the current project state and next experiment list, see `../status.md`.
